# 京东React Native App
本项目基于Redux架构，模仿京东客户端的主要页面，对于学习React Native, Redux有比较好的借鉴意义，希望对大家有所帮助。项目长期维护，会利用业余时间完善，欢迎大家star，fork（ps: 开始准备仿美团来着，发现美团和阿里系的数据都不是那么好抓了），

============

$ npm install
```

$ react-native run-ios/run-android
```
